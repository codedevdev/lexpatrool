**LexPatrol** — десктопный справочник для роли в **государственных службах** в многопользовательском RP (часто на базе GTA V: полиция, шериф, EMS и другие структуры — как принято на вашем сервере). Вы сами наполняете базу: кодексы, уставы и правила с сайтов или своих текстов. Дальше — быстрый поиск по статьям и **оверлей**: отдельное окно поверх игры с закреплёнными материалами.

Приложение **независимое**: не связано с Rockstar Games, издателями GTA и не «официальный» клиент какого-либо RP-сервера — это инструмент для хранения **ваших** импортированных материалов.

**Lex** — про нормы и право; **Patrol** — про патрульную работу.

---

## Что внутри

| Раздел | Зачем |
|--------|--------|
| **База знаний** | Документы и статьи в одном месте, полнотекстовый поиск. |
| **Импорт** | Добавить текст или страницу с сайта в базу — ваш источник правды по серверу. |
| **Читатель** | Удобное чтение статей, закладки, вынос нужного **на оверлей**. |
| **Оверлей** | Окно поверх игры: закрепы, поиск по базе, настройка прозрачности и горячих клавиш. |
| **Заметки** | Свои пометки и привязка к статьям. |
| **ИИ** | По желанию — подсказки по текстам из вашей локальной базы (настраивается в приложении). |
| **Настройки** | Горячие клавиши, оверлей, резервная копия базы, проверка обновлений. |

Есть вариант **portable**: можно держать программу и данные в одной папке и переносить между ПК.
