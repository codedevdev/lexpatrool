**Справочник норм и правил для ролевой игры в государственных организациях GTA5RP** (полиция / LSPD-style, шериф, EMS и др.). Десктопное приложение под **Windows**: локальная база статей, быстрый поиск и отдельное окно **оверлея** поверх игры (обычное окно ОС, не изменение клиента GTA5). Импорт — из ваших материалов или через встроенный браузер после входа на сайт.

<p align="center">
  <sub>Про имя: <strong>Lex</strong> — о нормах и праве; <strong>Patrol</strong> — патрульная и служебная работа.</sub>
</p>

## Сборки и релизы

Иконки под заголовком: **[релиз](https://github.com/codedevdev/lexpatrool/releases/latest)** — последняя опубликованная версия по тегам; **[сборка](https://github.com/codedevdev/lexpatrool/actions/workflows/release-windows.yml)** — статус workflow на `main`; **[скачивания](https://github.com/codedevdev/lexpatrool/releases)** — суммарное число загрузок всех файлов из Releases (данные GitHub).

Готовые **установщик (NSIS)** и **portable** `.exe` для Windows публикуются в [**Releases**](https://github.com/codedevdev/lexpatrool/releases/latest) — актуальная сборка на странице **Latest**.

Сборка и загрузка файлов в релиз выполняются **GitHub Actions** (workflow `.github/workflows/release-windows.yml`): после push тега вида `v1.1.0` (версию синхронизируйте с полем `version` в `package.json`) в Releases появится новая версия с установщиком и portable. Ручной запуск workflow (**Actions → Release (Windows) → Run workflow**) собирает проект и прикрепляет `.exe` как артефакты прогона (без создания GitHub Release).

## Возможности

| Область | Что делает приложение |
|--------|------------------------|
| **База знаний** | Документы и статьи в одном месте: дерево по документу, читатель, полнотекстовый поиск. |
| **Импорт** | Текст из буфера, HTML, встроенный браузер с ручным входом на сайт и импортом видимого содержимого; при необходимости — разбор по своим селекторам (DOM). |
| **Оверлей** | Отдельное окно поверх игры: закрепы статей, глобальный поиск по базе, переключение «клики в интерфейс / в игру», прозрачность и уровень «поверх других окон». |
| **Горячие клавиши** | Глобальные сочетания для показа оверлея, фокуса поиска и режима мыши — **можно переназначить** в настройках. |
| **Заметки и закладки** | Свои текстовые заметки (в т.ч. с привязкой к статье) и закладки по статьям из читателя. |
| **ИИ-помощник** | Контекстные ответы по материалам из базы (настраиваемые провайдеры и агенты). |
| **Резервное копирование** | Экспорт данных в JSON для переноса на другой ПК. |
| **Трей** | Закрытие главного окна сворачивает приложение в системный лоток; выход — из меню иконки. |

На части конфигураций доступна **portable-сборка**: данные могут храниться рядом с исполняемым файлом (удобно при переносе одной папкой).

## Безопасность и ограничения

- Нет обхода капчи, Cloudflare и прочих антибот-механизмов; закрытые страницы — только через обычный вход в браузере и импорт того, что вы видите на экране.
- Оверлей реализован как обычное окно операционной системы, **не** как модификация процесса игры.
- На Windows без коммерческой подписи возможны предупреждения SmartScreen — это ожидаемо для неподписанных сборок.

## Технологии

Electron, React, TypeScript, Vite, SQLite (в т.ч. FTS5), сборка под Windows через electron-builder.
